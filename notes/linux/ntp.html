Ntp

Utilities

ntpd

is an operating system daemon which sets and maintains the system time of day in synchronism with Internet standard time servers.

ntpdc

is a utility program used to query ntpd about its current state and to request changes in that state. It is deprecated, use ntpq instead.

Usage

Install and setup on Archlinux:

# pacman -S ntp

# # Run at backgroud
# systemctl start ntpd.service
# systemctl enable ntpd.service
or
# # Run at boot time
# systemctl start ntpdate.service
# systemctl enable ntpdate.service

Install and setup on CentOS:

# yum install ntp

# # Run at backgroud
# systemctl start ntpd.service
# systemctl enable ntpd.service
or
# # Run manually
# ntpdate 0.centos.pool.ntp.org
