Linux Memtest86+
================

Install memtest86+ on CentOS7: ::

    # yum install memtest86+
    # memtest-setup
    # grub2-mkconfig -o /boot/grub2/grub.cfg
    # reboot

Press F2 to force Multi-Threading (SMP).
