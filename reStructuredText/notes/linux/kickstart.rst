Kickstart
=========

In CentOS 7, plug in a usb, when load into boot menu page, type TAB key, append
following contents to kernel line:

::

    inst.ks=http://<ip1>:<port>/<ks.cfg> ip=<ip2>::<gateway>:<netmask>:<hostname>:<interface>:none
