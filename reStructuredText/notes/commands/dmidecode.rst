Dmidecode
=========

See *man dmidecode*

with *-t* option, either keyword or numbers (multiple numbers should be
splitted by comma) can be used.

Detect number of RAM slots used, speed, and size:

::

    # dmidecode -t 17

Detect bios info:

::

    # dmidecode -t bios
